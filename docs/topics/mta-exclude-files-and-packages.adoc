= Configuring {ProductShortName} to exclude packages and files


[id="ignored-locations_{context}"]
== Searching locations for exclusion

{ProductShortName} searches the following locations:

* `~/.{LC_PSN}/ignore/`
* `~/.windup/ignore/`
* `<{ProductShortName}_HOME>/ignore/`
